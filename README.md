# Food satiety explorer

An interactive graph of 232 common foods scored by how **filling they are per
calorie** (satiety) and sorted into 5 tiers. Search, filter by category / tier /
calories, and explore.

**Live:** https://meni-gottesman.github.io/satiety-explorer/

Static, self-contained (data baked in). Scores combine energy density, protein,
fibre, fat, food form and processing, validated against the Holt (1995) Satiety
Index.
