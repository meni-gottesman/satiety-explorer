# Unprocessed foods - satiety dot plot

Interactive dot plot of 2,338 truly unprocessed whole foods (Open Food Facts,
NOVA 1, English): x = energy density (kcal/g), y = fillingness per calorie
(0-100 percentile rank). Colour by tier or category, toggle per-category trend
lines, zoom/pan.

Live: https://meni-gottesman.github.io/satiety-explorer/
Paper: https://meni-gottesman.github.io/satiety-explorer/satiety_algorithm_paper.pdf
